UCP Explorer

A transparency tool for the Universal Commerce Protocol

This tool lets you see exactly what happens when an AI agent connects to a store's Universal Commerce Protocol (UCP) endpoints. Enter any domain below and the tool will:

  1. Fetch the UCP manifest at /.well-known/ucp to discover what services and capabilities the store advertises
  2. Connect to the MCP endpoint to list the tools the store makes available to AI agents (catalog search, cart management, checkout, etc.)
  3. Let you interact with those tools directly — search the catalog, ask about store policies, and view product details

Every request is shown with its URL, HTTP status, and raw JSON response so you can see the full back-and-forth between agent and merchant.

About UCP

What is the Universal Commerce Protocol?

UCP is an open protocol that defines how AI agents discover products, negotiate capabilities, process checkouts, and handle post-purchase interactions with merchants. It was co-developed by Google, Shopify, Etsy, Wayfair, Target, and Walmart, and is published under the Apache License 2.0.

What problem does UCP solve?

Without a standard, every AI agent would need custom integrations with every merchant. UCP provides a common language so that any agent can interact with any participating merchant through one protocol — handling product discovery, cart operations, checkout, and order management.

How does UCP work?

A merchant publishes a JSON manifest at /.well-known/ucp that describes the services and capabilities they support. This manifest includes the URL of their MCP (Model Context Protocol) endpoint. AI agents fetch this manifest, discover available tools (like catalog search or checkout), and interact with the merchant through standard JSON-RPC requests.

What is MCP in this context?

MCP (Model Context Protocol) is the transport layer that UCP uses for communication between agents and merchants. Merchants expose MCP endpoints that implement UCP capabilities as callable tools — for example, search_catalog, get_product_details, update_cart, and search_shop_policies_and_faqs.

What does this tool actually do?

This tool performs the same requests an AI agent would make when interacting with a store. It fetches the UCP manifest, connects to the MCP endpoint, lists available tools, and lets you call those tools interactively. All requests and responses are shown transparently so you can see exactly what data is exchanged.

Does every online store support UCP?

No. UCP is still in its early stages. Currently, Shopify stores have UCP support available through Shopify's platform. Other platforms may adopt UCP over time as the protocol matures.

Where can I learn more?

The official UCP specification is at ucp.dev. Shopify's implementation documentation is at shopify.dev/docs/agents.
